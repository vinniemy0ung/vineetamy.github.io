HTML

HTML is the standard markup language for creating Web pages.It describes the structure of the web page and it consists of series of elements.HTML elements tell the browser how to display the content.


CSS

CSS stands for Cascading Style Sheets.it is used to format the layout of a webpage.With CSS, you can control the color,font,size of text,spacing between elements,how elements are positioned and laid out,what background colors are to be used and much more!


JAVASCRIPT

JavaScript is a high-level dynamic interpreted programming language.It allows client-side scripting to create completely dynamic web applications and websites.JavaScript was initially designed for making pages "alive".It can be executed on the browser as well as the server.There are languages that get "transpiled" to JavaScript.
